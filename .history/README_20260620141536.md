# ❤️ Heart Disease Prediction App

A simple machine learning web application that predicts the likelihood of heart disease based on patient health information.

The application is built using **Python**, **Scikit-learn**, and **Streamlit**. Users can enter medical parameters such as age, blood pressure, cholesterol level, ECG results, and other health indicators to receive a prediction from a trained KNN model.

---

## Features

* User-friendly web interface built with Streamlit
* Real-time heart disease prediction
* Trained K-Nearest Neighbors (KNN) model
* Input validation through sliders and dropdowns
* Data preprocessing using a saved scaler
* Clean and responsive dashboard

## Input Parameters

The model uses the following patient information:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise Induced Angina
* Oldpeak
* ST Slope

## Run Locally

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## Model Information

The prediction model was trained using the K-Nearest Neighbors (KNN) algorithm. Input features are scaled using a saved StandardScaler before being passed to the model for prediction.
